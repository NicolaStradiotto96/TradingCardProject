# MY TRADING CARD WEBSITE

## PROJECT DESCRIPTION

This project focuses on the development of an **online platform for buying and selling collectible cards**.
The website will allow users to browse, search, and purchase cards from different categories, such as trading card games.
The platform will feature secure payments, user accounts, a product rating system, and a responsive design to ensure a smooth experience on both desktop and mobile devices.
The goal is to create a safe, fast, and engaging marketplace for collectors and enthusiasts.

## LANGUAGES AND FRAMEWORKS

- HTML
- CSS
- JavaScript
- Bootstrap
